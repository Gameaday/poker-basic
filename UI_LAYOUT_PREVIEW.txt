   - Smooth hover effects and scaling
   - Selection highlighting with gold border
   - Transition effects between screens

                             ARCHITECTURE
===============================================================================

┌─ PRESENTATION LAYER ──────────────────────────────────────────────────────┐
│ PokerFXApplication → GameView → CardView, SettingsDialog                 │
└───────────────────────────────────────────────────────────────────────────┘
┌─ MANAGEMENT LAYER ────────────────────────────────────────────────────────┐
│ GameSettingsManager ← → ThemeManager ← → InputHandler                    │
└───────────────────────────────────────────────────────────────────────────┘
┌─ GAME LOGIC LAYER ────────────────────────────────────────────────────────┐
│ Existing game logic (Main, Player, GameEngine, etc.) - PRESERVED         │
└───────────────────────────────────────────────────────────────────────────┘

This modern JavaFX interface replaces the old Swing GUI while maintaining 
complete backward compatibility and all existing functionality!
